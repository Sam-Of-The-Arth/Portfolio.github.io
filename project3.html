Motion Tracking Glove

Hardware Interface

Overview

A wearable glove capable of detecting finger motion and transmitting real-time gesture data. Tuned firmware ensures accurate tracking, noise reduction, and full-bend activation thresholds.

Key Features

  • Individual finger motion tracking
  • Calibrated flex sensor readings
  • Gesture debouncing system
  • Low-latency data transmission
  • Firmware-tuned full-bend detection
  • Lightweight wearable hardware

Technologies Used

Arduino ESP32 Flex Sensors C++ Bluetooth / Serial

Project Details

The glove’s firmware includes calibration, filtering, debouncing, and gesture mapping. It supports smooth control input suitable for robotics, games, and interactive systems. Sensitivity was tuned so only full bends register as gestures.
